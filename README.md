# Zomato Data Analysis & Visualization (DAV)

This project performs exploratory data analysis and visualization on the Zomato restaurant dataset.

## Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn
- Google Colab

## Project Features
- Data cleaning and preprocessing
- Analysis of restaurant ratings, locations, cuisines
- Visualizations for better insights
